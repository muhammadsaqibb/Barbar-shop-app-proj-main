# **App Name**: BookItEasy

## Core Features:

- User Authentication: Implement user registration, login, and logout using Firebase Authentication with email and password.
- Appointment Booking: Allow clients to book appointments by selecting a service, date, and time.
- Appointment Storage: Store appointment details (clientId, service, date, time, status, createdAt) in Firestore with a default status of 'pending'.
- User Role Management: Store user data (name, email, role) in a 'users' collection in Firestore.
- My Appointments View: Display a list of appointments booked by the logged-in user.

## Style Guidelines:

- Primary color: Deep sky blue (#42A5F5) to convey trust and professionalism, inspired by the reliability needed for bookings.
- Background color: Very light blue (#F0F4F8), nearly white to ensure a clean and calming backdrop that supports readability.
- Accent color: A shade of spring green (#9CCC65), used for highlighting interactive elements and conveying a sense of growth.
- Font: 'PT Sans', a humanist sans-serif suitable for both headlines and body text.
- Use a clean and straightforward layout to focus on functionality, making the booking process intuitive.
- Employ simple and easily recognizable icons to represent different services and appointment status.
- Implement subtle animations, such as smooth transitions when navigating between screens, to enhance user experience.
